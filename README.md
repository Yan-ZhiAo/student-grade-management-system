# 学生成绩管理系统

学生成绩管理系统是一个基于 C 语言链表结构实现的控制台管理项目。系统围绕学生基本信息和课程成绩管理展开，支持管理员登录、学生信息录入、增删改查、单科成绩排名、按学号查询成绩，以及筛选不及格科目较多的学生名单。项目使用本地文件保存学生数据，适合展示 C 语言结构体、链表、文件读写和菜单式交互的综合实践。

## 项目功能

- 管理员登录：启动系统后输入管理员密码，验证通过后进入功能菜单。
- 信息录入：支持批量录入学生学号、姓名，并分别录入数学、英语、专业课成绩。
- 信息维护：可以新增、删除、修改指定学生的基础信息和各科成绩。
- 信息展示：按链表顺序展示当前系统中的学生成绩记录。
- 成绩查询：输入学号后查询对应学生的数学、英语和专业课成绩。
- 成绩排名：分别按数学、英语、专业课成绩从高到低输出排名。
- 学业预警：筛选不及格科目达到 2 科及以上的学生名单。
- 文件持久化：通过 `Student.sys` 保存和读取学生信息，关闭程序后仍可保留数据。

## 技术栈

| 模块 | 技术 |
| --- | --- |
| 编程语言 | C |
| 数据结构 | 结构体, 单向链表 |
| 数据存储 | 本地二进制文件 |
| 交互方式 | Windows 控制台菜单, `conio.h` |
| 开发环境 | Dev-C++ 5.11, MinGW |
| 工程文件 | Dev-C++ `.dev` 工程, `Makefile.win` |

## 项目结构

```text
.
|-- main.c              # 程序入口和主菜单调度
|-- main.h              # 学生结构体、链表结点和函数声明
|-- Menu Login.c        # 菜单界面和管理员登录
|-- Input.c             # 学生基础信息与成绩录入
|-- Add.c               # 新增学生信息
|-- Delete.c            # 删除学生信息
|-- Modify.c            # 修改学生信息
|-- Show Search.c       # 展示全部学生信息和按学号查询
|-- Grade.c             # 数学、英语、专业课成绩排名
|-- Failed.c            # 统计不及格科目较多的学生
|-- Save Read.c         # Student.sys 文件保存和读取
|-- Makefile.win        # Dev-C++ 生成的构建脚本
|-- 项目1.dev           # Dev-C++ 工程文件
|-- 测试数据.txt        # 手工测试用学生数据
|-- docs/
|   |-- ARCHITECTURE.md # 系统结构和数据流说明
|-- README.md
```

## 快速开始

### 1. 使用 Dev-C++ 打开

1. 打开 Dev-C++。
2. 选择 `项目1.dev`。
3. 编译并运行项目。

程序启动后输入管理员密码：

```text
123
```

### 2. 使用 Makefile.win 构建

如果本机已安装 Dev-C++ 对应的 MinGW 环境，可以在项目目录执行：

```powershell
mingw32-make -f Makefile.win
```

生成的可执行文件为 `项目1.exe`。编译产物不会提交到 Git。

## 数据说明

程序运行时会在当前目录生成 `Student.sys`，用于保存学生结构体数据。该文件属于本地运行数据，已加入 `.gitignore`，需要重新测试时可以删除后再运行程序。

学生成绩包含三门课程：

| 字段 | 说明 |
| --- | --- |
| `math` | 数学成绩 |
| `english` | 英语成绩 |
| `cs` | 专业课成绩 |

## 验证

本项目依赖 Windows 控制台和 `conio.h`，建议在 Dev-C++ 或 Windows MinGW 环境下验证。基础验证流程如下：

1. 输入管理员密码 `123` 进入系统。
2. 使用菜单 `1` 录入学生基本信息和课程成绩。
3. 使用菜单 `5` 查看学生列表。
4. 使用菜单 `6` 查看三门课程成绩排名。
5. 使用菜单 `7` 按学号查询学生成绩。
6. 使用菜单 `8` 查看不及格科目达到 2 科及以上的学生名单。

更完整的模块关系和数据流见 [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md)。
